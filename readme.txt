Instrucciones para clonar el proyecto

Ingresar a mi GitHub: https://github.com/juangiraldor

Se deben descargar todos los archivos del repositorio.
Los scripts de automatización se encuentran en lenguaje JAVA por lo cual se debe tener instalado
en la máquina el Oracle OpenJDK 21.0.2 de JAVA para poder ejecutarse.
El proyecto está construido sobre MAVEN.
También es importante tener en cuenta, la versión del navegador Chrome que se debe usar es la 133.
Dentro de los archivos se encuentra el chromedriver.exe para esta versión del navegador.

Los casos de prueba se realizaron de acuerdo a cada uno de los pasos descritos en el PDF,
los cuales se pueden encontrar en la siguiente ruta: prueba-DoubleVPartners/src/test/java/com.doublev
El caso de prueba CompleteProcess es el que contiene el flujo completo de compra.

url para clonar el repositorio: https://github.com/juangiraldor/prueba-DoubleVPartners.git
Mediante GitHub CLI: gh repo clone juangiraldor/prueba-DoubleVPartners
